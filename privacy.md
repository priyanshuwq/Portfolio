# Privacy Policy

**Last Updated:** March 1, 2026

## Overview

This portfolio website is designed with privacy in mind. I collect minimal data necessary to provide functionality and improve user experience. No personal information is sold or shared with third parties for marketing purposes.

## Information Collection and Use

### 1. Visitor Counter
- **What:** A simple counter that increments when you visit the site
- **How:** Stored using JSONbin.io (third-party service) in production, or local file storage in development
- **Why:** To display site popularity
- **Data Stored:** Only the total visit count and last update timestamp
- **Personal Information:** None - no IP addresses, locations, or identifying information is collected

### 2. Spotify Integration
- **What:** Displays currently playing or recently played music
- **How:** Fetches from Spotify Web API using my personal OAuth token
- **Why:** To share what I'm listening to
- **Data Collected:** None from visitors - this is my personal Spotify data displayed publicly
- **Your Privacy:** No visitor data is sent to Spotify

### 3. GitHub Contributions
- **What:** Displays my public GitHub contribution calendar
- **How:** Fetches from GitHub's public API
- **Why:** To showcase coding activity
- **Data Collected:** None from visitors - this is my public GitHub data
- **Your Privacy:** No visitor data is sent to GitHub

### 4. AI Chatbot
- **What:** Interactive chatbot to answer questions about my portfolio
- **How:** Powered by Groq AI (llama-3.1-8b-instant model)
- **Why:** To provide an interactive way to learn about my work
- **Data Collected:** Messages you send are processed by Groq AI but are not stored permanently
- **Data Retention:** Conversation data is only retained during your session
- **Privacy:** Messages are processed server-side and sent to Groq AI API for inference only

### 5. Third-Party Services

#### Google Fonts
- **Purpose:** Loading custom web fonts (Hanken Grotesk)
- **Privacy:** Google may collect technical information like IP address for font delivery
- **Learn More:** [Google Fonts Privacy Policy](https://developers.google.com/fonts/faq/privacy)

#### JSONbin.io
- **Purpose:** Storing visitor counter in production
- **Privacy:** Only stores anonymous visit count, no personal data
- **Learn More:** [JSONbin.io Privacy Policy](https://jsonbin.io/privacy-policy)

#### Groq AI
- **Purpose:** Powering the AI chatbot
- **Privacy:** Your messages are processed for inference only
- **Learn More:** [Groq Privacy Policy](https://groq.com/privacy-policy/)

## Data Storage

### Local Storage
- **Theme Preference:** Your light/dark mode preference is stored in browser local storage
- **Purpose:** Persist your theme choice across visits
- **Control:** You can clear this by clearing browser local storage

### Cookies
This website does not use cookies for tracking or analytics.

## Analytics
This website does **not** use Google Analytics, Facebook Pixel, or any other tracking/analytics services. The only "tracking" is the basic visitor counter mentioned above.

## Data Sharing
